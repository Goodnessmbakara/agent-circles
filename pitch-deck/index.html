The Problem
01 / 08

1.7 billion people save through rotating circles. Too many fail.

Trust Failure

A member collects the pot early and disappears. No enforcement mechanism. The group loses money and trust collapses.

Coordination Failure

The organizer manually tracks payments, chases late contributors, calculates schedules. Unpaid labor that leads to burnout.

Known as ajo (Nigeria), susu (Ghana), tanda (Mexico), chit fund (India), hagbad (Somalia) — the oldest savings mechanism, still running on trust and spreadsheets.

The Solution
02 / 08

Smart contracts for enforcement. AI agents for coordination.

On-Chain Rules

A Soroban smart contract locks contributions, enforces rotation order, and auto-distributes payouts. No one can run with the pot.

How It Works
03 / 08

The ROSCA Cycle

01

Create

Set amount, period, members

02

Join

Members fill the pool

03

Contribute

Everyone pays each round

04

Payout

One member gets the pot

05

Rotate

Repeat for all members

Example: 5 members × 10 USDC = 50 USDC pot. Each pays 50 total, receives 50 once (minus agent fee). Forced savings, not lending.
