# Agent Circles Brand Guide

## Brand Identity
Agent Circles is an on-chain rotating savings and credit association (ROSCA) platform built on Stellar. The brand identity reflects security, automation (agents), and community (circles).

## Typography

### Primary Font: **Outfit**
*Alternative: Inter*

**Outfit** is a geometric sans-serif that feels modern, approachable, and tech-forward.

- **Headings**: SemiBold or Bold (tracking: -0.02em)
- **Body**: Regular or Medium
- **Captions/Labels**: Medium (all caps, tracking: 0.1em)

### Usage Example
```css
font-family: 'Outfit', sans-serif;
```

## Logo & Iconography

### The "Mark"
The Agent Circles mark consists of two geometric primitives:
1.  **The Slanted Block (Parallelogram)**: Represents the "Agent" or the movement/process.
2.  **The Circle**: Represents the "Circle" or the pool/community.

### Logo Variants
- **Full Logo**: Mark + "AGENT CIRCLE" text in all-caps Outfit.
- **Icon Only**: The mark alone, centered in a square or circular container.

### Construction
- The mark should maintain a 1:1 aspect ratio.
- The gap between the block and the circle should be consistent (approx 10% of the circle's diameter).

## Visual Language
- **Gradients**: Subtle violet-to-indigo gradients for depth.
- **Glassmorphism**: Use `backdrop-filter: blur()` for overlays.
- **Micro-animations**: Smooth transitions (150ms-200ms) for hover states.
